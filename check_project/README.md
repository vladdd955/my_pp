# my_pp

# Стандартные команды для работы с проектом
```git clone https://github.com/vladdd955/my_pp.git```

```composer install```

```npm install```

```npm install laravel```

```npm run dev```

```php artisan migrate```

```php artisan serve``` запуск проекта в веб-режиме

```postman_import_file``` раскладку для теста API, отправил с таском.
